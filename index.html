ABOUT

adrenaline is an unblocked game website created so that I can play games in school.

games are loaded from the /games/ folder. each game lives in its own subdirectory:

games/
  └── my-game/
      ├── thumbnail.png
      ├── game.html
      └── data.json

data.json: { "title": "...", "genre": "...", "added": "2024-01-01" }

run node generate-manifest.js to auto-build games/manifest.json.
